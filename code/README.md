# Cloned Repositories

## Repo 1: MMLongBench
- **URL**: https://github.com/EdinburghNLP/MMLongBench
- **Purpose**: Official implementation and dataset scripts for MMLongBench (Multimodal Long-Context Benchmark).
- **Location**: `code/MMLongBench/`
- **Key Files**:
  - `scripts/download_text_data.sh`: Download dataset annotations.
  - `eval.py`: Evaluation script.
  - `vlm_model/`: Model wrappers.
- **Notes**: Contains scripts to download the dataset used in this project.

## Repo 2: Recursive Visual Programming (RVP)
- **URL**: https://github.com/para-lost/RVP
- **Purpose**: Implementation of Recursive Visual Programming for VQA/Reasoning.
- **Location**: `code/RVP/`
- **Key Files**:
  - `models/`: Model definitions (likely BLIP/Llava based).
  - `inference.py` (or similar, check repo structure).
- **Notes**: Implements the recursive logic proposed in the hypothesis.

## Repo 3: DINOv (Visual In-Context Learning)
- **URL**: https://github.com/UX-Decoder/DINOv
- **Purpose**: Official implementation of "Visual In-Context Learning" (CVPR 2024).
- **Location**: `code/DINOv/`
- **Key Files**:
  - `train_net.py`: Main training script.
  - `demo/`: Demonstration scripts.
- **Notes**: Provides the Visual In-Context Learning mechanism to be integrated.
